# 项目需求：外贸批发跳绳官网

## 目标
开发一个用于展示和销售跳绳的英文官网，面向海外批发采购客户。网站风格参考 Apple 官网：大留白、大图、滚动动画、产品细节特写、强品牌感。核心目标不是零售下单，而是获取批发询盘。

技术建议：Next.js + Tailwind CSS + GSAP ScrollTrigger。Next.js 适合做 React 全栈官网，GSAP ScrollTrigger 适合做滚动触发动画，部署优先考虑 Vercel。参考官方文档：Next.js、GSAP ScrollTrigger、Vercel。  

## 技术栈
- Next.js App Router
- TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger
- Framer Motion 可选
- 表单提交：先做静态 mock，预留 API
- 部署：Vercel

## 页面语言
英文为主，面向海外客户。

## 网站风格
关键词：
- Apple-like
- Premium sports brand
- Minimal

- Smooth scroll animation
- Wholesale / B2B professional

不要做成传统阿里巴巴风格，不要满屏文字，不要廉价促销感。

## 动画要求
使用 GSAP ScrollTrigger：
- 首屏产品淡入
- 滚动时文字淡入上移
- 产品细节图轻微缩放
- 数字/参数出现动画
- 不要过度复杂，保证移动端流畅

GSAP ScrollTrigger 用于滚动触发、pin、scrub 等动画场景；注意避免写太重的动画，移动端优先性能。

## 重要要求
1. 不要做成普通电商模板。
2. 不要堆很多廉价促销文案。
3. 整体要像高端运动科技品牌。
4. 页面核心是获取批发询盘。
5. CTA 要反复出现：Get Wholesale Quote。
6. 先用占位图片，但代码结构要方便后续替换真实产品图。
7. 代码要干净，组件化，方便继续扩展。
8. 所有文案先用英文。
9. 页面要能直接运行。
10. 最终给出启动命令和部署说明。
